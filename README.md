# Digi-Foss
Bot supervisor and contributor to admins in Group Management

## Feature

Digi-FOSS Bot Features

#### Delete Message

Delete message by number 

```
/del [number]
```

#### Spam

Spamming 

```
/spam [number]
```

#### ramadan

Get the remaining time until adhan 

```
/ramadan [city name] (Default: Tehran)
```
